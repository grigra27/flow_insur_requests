# Функциональность "Резюме по страховой сделке"

## Описание

Страница "Резюме по страховой сделке" предоставляет структурированное представление всей информации о завершенной страховой сделке. Данные на этой странице соответствуют информации, которая выгружается на технический лист при генерации Excel файла свода.

## Доступ к функциональности

Страница резюме доступна только для сводов со статусом **"Завершен: акцепт/распоряжение"** и с выбранной страховой компанией.

### Способы доступа:

1. **Из списка сводов**: Нажать на ссылку "резюме" рядом с названием выбранной компании
2. **Из детальной страницы свода**: Нажать кнопку "Резюме по сделке" в блоке с выбранной СК

## Отображаемая информация

### 1. Информация о сделке
- Номер заявки (ДФА)
- Название клиента
- ИНН клиента
- Филиал
- Выбранная страховая компания
- Статус свода
- Дата создания

### 2. Параметры страхования
- Тип страхования
- Предмет лизинга (информация о ТС)
- Банк-кредитор
- Тип франшизы
- Наличие рассрочки
- Цели использования

### 3. Дополнительные параметры для КАСКО/Спецтехники
(Отображается только для соответствующих типов страхования)
- Наличие автозапуска
- Комплектность ключей
- ПТС/ПСМ
- Телематический комплекс

### 4. Дополнительные параметры для страхования имущества
(Отображается только для страхования имущества)
- Территория страхования
- Условие перевозки
- Условие СМР (строительно-монтажные работы)

### 5. Предложения выбранной компании
Таблица со всеми предложениями от выбранной страховой компании:
- Год страхования
- Страховая сумма
- Франшиза-1 и премия с франшизой-1
- Франшиза-2 и премия с франшизой-2
- Условия рассрочки для каждого варианта
- Комментарии к предложениям

Для многолетних предложений автоматически рассчитывается итоговая сумма за все годы.

### 6. Примечания к своду
Дополнительные примечания, если они были добавлены к своду.

## Функциональность

### Печать
Страница оптимизирована для печати:
- Кнопка "Печать" в правом верхнем углу
- При печати скрываются элементы навигации
- Сохраняется цветовое оформление для лучшей читаемости

### Навигация
- Кнопка "К своду" для возврата к детальной странице свода

## Примеры использования

### Пример 1: Просмотр резюме однолетней сделки

1. Открыть список сводов
2. Найти свод со статусом "Завершен: акцепт/распоряжение"
3. Нажать на ссылку "резюме" рядом с названием выбранной компании
4. Просмотреть все параметры сделки и предложение от выбранной компании

### Пример 2: Печать резюме многолетней сделки

1. Открыть страницу резюме
2. Проверить, что отображаются все годы предложения
3. Проверить итоговую сумму за все годы
4. Нажать кнопку "Печать"
5. Выбрать принтер или сохранить как PDF

### Пример 3: Доступ из детальной страницы свода

1. Открыть детальную страницу свода
2. В блоке "Управление статусом" найти информацию о выбранной СК
3. Нажать кнопку "Резюме по сделке"
4. Просмотреть полную информацию о сделке

## Преимущества

1. **Единое представление**: Вся информация о сделке в одном месте
2. **Соответствие Excel**: Данные идентичны техническому листу
3. **Удобство печати**: Оптимизированный вид для печати
4. **Быстрый доступ**: Прямые ссылки из списка и детальной страницы
5. **Адаптивность**: Корректное отображение на разных устройствах
6. **Контекстность**: Показываются только релевантные поля в зависимости от типа страхования
