Цветовое кодирование:
  • Красным — ячейки, отличающиеся для ИП
  • Жёлтый фон — строки с различиями между типами заявок
  • Зелёный фон — новые параметры
Смещение строк для ИП: для заявок от ИП применяется +1 для всех строк > 8.
Юр.лица: строки без изменений. Пример: ИНН в D9, франшиза в D29.
ИП: строки 9+ смещены на +1. Пример: ИНН в D10, франшиза в D30.

КАСКО/спецтехника — юр.лицо
Стандартная структура файла:
D7: ООО "АЛТЫН ЯР"
D9: 1234567890 (ИНН 10 цифр)
D21: КАСКО
N17: ✓ (1 год)
D29: Без франшизы (none)
F34: (пустая = есть рассрочка)
M24: да (есть автозапуск)
C45: ✓ КАСКО C/E
КАСКО/спецтехника — ИП
Структура с дополнительной строкой 8:
D7: ИП Иванов И.И.
D8: +1 СТРОКА
D10: 123456789012 (ИНН 12 цифр)
D22: КАСКО (смещено)
N18: ✓ (1 год)
E30: 50000 (with_franchise)
M25: нет (НЕТ автозапуска)
D46: ✓ КАСКО C/E
Имущество — юр.лицо
Специальная логика для имущества:
D7: ООО "СТРОЙКОМ"

C48: (пустая, СМР = FALSE)
L20: Москва и МО Территория

C49: СМР СМР = TRUE

Внимание! Неправильный выбор формата или типа заявки приведёт к извлечению данных из неправильных ячеек.
Ключевые особенности:
  • Смещение для ИП: применяется только к строкам > 8
  • Информация об имуществе: смещение НЕ применяется (строки 42/43 уже учитывают тип заявки)
  • Год выпуска (КАСКО/спецтехника): столбец J, строки 43/45/47/49
  • Статус имущества (КАСКО/спецтехника): столбец K, строки 43/45/47/49
  • Новые параметры: КАСКО C/E (строка 45), перевозка (C44), СМР (C48)
  • Перевозка и СМР определяются только для формата «Имущество»
